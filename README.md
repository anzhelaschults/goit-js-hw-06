# goit-js-hw-06

Homework assignment for GoIT JavaScript course. Topic 10: OOP. Classes.
Домашнє завдання курсу JavaScript від GoIT. Тема 10: ООП. Класи.

## Tasks / Завдання

- **Task 1** — `customer` object: refactoring of object methods by adding missing this references to properties / рефакторинг методів об'єкта з додаванням відсутніх this при зверненні до властивостей
- **Task 2** — `Storage` class: manages a warehouse of products with private property items and methods getItems(), addItem(), removeItem() / клас для управління складом товарів з приватною властивістю items та методами getItems(), addItem(), removeItem()
- **Task 3** — `StringBuilder` class: builds strings with private property value and methods getValue(), padEnd(), padStart(), padBoth() / клас для конструювання рядків з приватною властивістю value та методами getValue(), padEnd(), padStart(), padBoth()
